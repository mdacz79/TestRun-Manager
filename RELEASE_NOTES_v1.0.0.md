## Instalacja

### Automatyczna instalacja

**Linux/Mac:**
```bash
tar -xzf testrun-manager-v1.0.0.tar.gz
cd testrun-manager-v1.0.0
chmod +x install.sh
./install.sh
```

**Windows:**
```powershell
Expand-Archive testrun-manager-v1.0.0.zip
cd testrun-manager-v1.0.0
.\install.ps1
```

### Ręczna instalacja

Zobacz plik `INSTALLATION.md` w archiwum.

## Wymagania systemowe

- **Node.js**: v18.0 lub nowszy
- **RAM**: 2 GB (zalecane 4 GB)
- **Dysk**: 1 GB wolnego miejsca
